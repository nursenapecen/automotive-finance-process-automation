# Automotive Finance Process Automation

Business Analysis Case Study

---

## Project Overview

This project demonstrates the digital transformation of an automotive financing process.

The goal is to reduce repetitive manual work performed by finance advisors by introducing OCR and Robotic Process Automation (RPA).

The solution enables customer information to be entered once and automatically submitted to multiple finance institutions.
