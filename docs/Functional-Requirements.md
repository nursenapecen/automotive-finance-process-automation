# Functional Requirements

## 1. Authentication & Authorization

FR-001

The system shall authenticate finance advisors using corporate credentials.

FR-002

The system shall support multi-factor authentication (OTP) for finance institution logins.

FR-003

The system shall verify user authorization before allowing loan application submission.

FR-004

The system shall log all user login and logout activities.

FR-005

The system shall restrict application approval to authorized user roles.


## 2. Customer Management

FR-006

The system shall allow finance advisors to create a new loan application.

FR-007

The system shall capture customer identity information.

FR-008

The system shall capture customer contact information.

FR-009

The system shall capture customer address information.

FR-010

The system shall capture customer occupation information.

FR-011

The system shall capture customer income information.

FR-012

The system shall allow editing customer information before submission.

FR-013

The system shall validate mandatory customer information.

FR-014

The system shall prevent duplicate customer applications.


## 3. Vehicle Management

FR-015

The system shall capture vehicle information.

FR-016

The system shall support both new and second hand (used) vehicle financing.

FR-017

The system shall retrieve vehicle variant information.

FR-018

The system shall display invoice amount.

FR-019

The system shall display annual maintenance amount.

FR-020

The system shall support tax exemption selection.

FR-021

The system shall validate vehicle information before submission.


## 4. OCR Integration

FR-022

The system shall allow document upload.

FR-023

The system shall extract customer information using OCR.

FR-024

The system shall extract address information from uploaded documents.

FR-025

The system shall extract identity document information.

FR-026

The system shall extract vehicle information from uploaded documents.

FR-027

The system shall allow manual correction of OCR results.

FR-028

The system shall validate OCR confidence before submission.

FR-029

The system shall store uploaded documents.


## 5. Finance Institution Selection

FR-030

The system shall allow users to select one or more finance institutions.

FR-031

The system shall support individual loan applications.

FR-032

The system shall support commercial loan applications.

FR-033

The system shall store selected finance institutions.


## 6. Loan Application

FR-034

The system shall capture requested loan amount.

FR-035

The system shall capture loan maturity.

FR-036

The system shall capture payment type.

FR-037

The system shall capture loan type.

FR-038

The system shall allow campaign selection.

FR-039

The system shall display available campaigns.

FR-040

The system shall display repayment plans.

FR-041

The system shall allow advisors to review loan information before submission.


## 7. RPA Automation

FR-042

The system shall trigger the RPA process after advisor confirmation.

FR-043

The RPA bot shall log into selected finance institution portals.

FR-044

The RPA bot shall populate customer information automatically.

FR-045

The RPA bot shall populate vehicle information automatically.

FR-046

The RPA bot shall upload required documents.

FR-047

The RPA bot shall submit loan applications.

FR-048

The RPA bot shall capture application responses.

FR-049

The RPA bot shall update application status.

FR-050

The RPA bot shall retry failed submissions.


## 8. Loan Status Tracking

FR-051

The system shall assign a unique application ID.

FR-052

The system shall display application status.

FR-053

The system shall display finance institution responses.

FR-054

The system shall display missing document notifications.

FR-055

The system shall display approval and rejection reasons when available.

FR-056

The system shall display application history.


## 9. Reporting

FR-057

The system shall provide an application list.

FR-058

The system shall filter applications by finance institution.

FR-059

The system shall filter applications by branch.

FR-060

The system shall filter applications by date.

FR-061

The system shall export reports.

FR-062

The system shall display processing statistics.


##10. Notifications

FR-063

The system shall notify finance advisors when application status changes.

FR-064

The system shall notify customers after application completion.

FR-065

The system shall notify advisors about missing documents.

FR-066

The system shall send email notifications.

FR-067

The system shall support in-system notifications.


## 11. Audit & Logging

FR-068

The system shall record all RPA activities.

FR-069

The system shall record user actions.

FR-070

The system shall record application history.

FR-071

The system shall maintain audit logs for compliance purposes.


## 12. Security

FR-072

The system shall encrypt sensitive customer information.

FR-073

The system shall protect customer data during transmission.

FR-074

The system shall restrict access based on user roles.

FR-075

The system shall comply with applicable data protection regulations.
